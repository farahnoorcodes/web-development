Text Styling

1. font-family

font-family sets the typeface for text. Key things to know:

p {
  font-family: 'Poppins', Arial, sans-serif;
}

How it works — the fallback stack: You list fonts in order of preference, separated by commas. The browser uses the first one it can find/load, and falls back to the next if it's unavailable. Always end with a generic family as a safety net.

Generic families (always include one at the end):
serif — has small strokes at letter ends (e.g., Times New Roman)
sans-serif — clean, no strokes (e.g., Arial, Helvetica)
monospace — fixed-width, all characters same size (e.g., Courier New) — good for code
cursive — script-like, handwriting style
fantasy — decorative

Quoting rule: font names with spaces need quotes — 'Open Sans', but single-word names don't need them — Arial.

Using custom/Google Fonts:

using html:

<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;700&display=swap" rel="stylesheet">

using css:

body {
  font-family: 'Poppins', sans-serif;
}

Or self-hosted with @font-face:

@font-face {
  font-family: 'MyFont';
  src: url('myfont.woff2') format('woff2');
}
body {
  font-family: 'MyFont', sans-serif;
}
