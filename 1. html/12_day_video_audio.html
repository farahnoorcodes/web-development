Audio and Video


Some important attributes: 
            src = source (short form) :
            -> Tells the browser where the file is located — the path or URL to your audio/video file.
            -> Example: src="hooligan.mp3" means "look for a file called hooligan.mp3 in this same folder."
            type = tells the browser what format/type the file is, so it knows how to handle it.
            -> type="audio/mpeg" → this is an MP3 audio file
            -> type="video/mp4" → this is an MP4 video file

Audio

you can add audio in your web page using <audio> tag.

steps of adding audio:

  • Place your audio file in the same folder as your HTML file (or note its subfolder path).
  • first add audio tag with "controls" written in the opening tag
  • second, add source tag inside the audio tag
  • add src and type attributes inside source — tells the browser where the file is and what format it is
  • Fallback text__If you want to type something in the case browser unable to load your file then type msg in audio tag that only appear when file doesn't load

Video

you can add video in your webpages using <video> tag.

step of adding video

  • Place your video file in the same folder as your HTML file (or note its subfolder path).
  • Use the <video> tag with "controls" written inside opening tag
  • add <source> inside video tag
  • add src and type attributes inside source — tells the browser where the file is and what format it is
  • Fallback text — shows only if the browser can't play <video> at all

key attributes :

  • control-> Play/pause/volume controls
  • autoplay-> Plays automatically, use with mute to avoid interruption
  • loop-> Repeats the video
  • mute-> Starts with no sound
  • poster ="thumbnail.jpg"-> Shows an image before the video plays
  • preload-> Preload set to load only metadata (like duration), not the whole file, until user hits play:
